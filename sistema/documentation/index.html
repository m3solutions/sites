Description - top

This application is your very own leads and contacts manager. This tool allows you to keep track of your leads and / or contacts, organize them by Source, Type, or Status so that you can readily view your contacts and track them. This system is set up to allow multiple users to manage your leads and contacts. A log keeps track of detailed changes made by all users which is paginated, searchable and tracks user ip addresses so you know who changed what. Logins have some extra security built in to include sha1 client side hashing along with server side salt, and all ajax requests and forms require valid session tokens. There are 2 user roles - admin and user. Admins have access to the settings tab which lets them change all configurable items (including adding, removing, and modifying users). Normal users can edit, add, delete leads, and add or remove notes for the leads. LCM is highly configurable for your needs, allowing you to create your own Source Groups, Type groups and Status groups. Graphical representations on the Statistics page let you quickly see your top Lead Sources, Lead Types, Status counts and trends in lead generation in the last 12 month time period.

  1. Create your own Source, Type, and Status groups
  2. Import / Export leads in csv, xls or xlsx format.
  3. Support built in for results pagination, searching, sorting.
  4. Installation script to help you get going, if you can install wordpress you can install LCM
  5. Dynamic Statistics and graphs to show important information
  6. Multiple Users
  7. Multiple notes can be created for each lead
  8. Leads can have a primary email address and up to 3 secondary email addresses
  9. Admins can manage other user accounts
  10. Lead creation data and modification tracking built in
  11. Add, Remove, Edit leads, and create Notes for them.
  12. Changes to leads and contacts are highlighted for quick reference point.
  13. Quickly remove all Leads (If you really need to!) in the settings area
  14. No Limit on number of Sources or Types, Status is limited by space in top banner

Quick Start - Install - top

To install, extract the zip file on your webserver taking note of the directory you put it in (default folder created is lcm). Next you will need to create a new MySql Database with your favorite program and take note of the database name, your username and password used. In phpMyAdmin, here's what creating a database looks like:

Once you have created the database, open your browser to www.yoursite.com/lcm/ (or the folder you put it in) where you should be presented with the following screen:

Insert your database settings, and click 'Test Database', if the installer is able to connect to the database you will be presented with the following screen

At this point just click the 'Run Install' button which will create the tables in your database and config file for you. LCM will give you the option of downloading the config file (variables.php) in case it is unable to write the file, and it's a good idea to download this file anyways since it contains your settings. To download, click the yellow button that says 'Download Variables File'. If LCM could not write the file all you would need to do in this case is upload the file to your server in the classes directory. Here is what that screen looks like:

At this time, you can click the 'Login to LCM' and begin working in it. If you are redirected to the install page again, that means the variables.php file was not written, so you'll need to copy it to the server. Otherwise you will be presented with the login screen to your fresh install of LCM:

Core Files and Structure - top

This is an explanation of the files included and their purpose.

root folder
  1. config.php - This is the Settings page where you set up your groups, users, view access log, export/import leads etc..
  2. footer.php - This is the footer file for all pages
  3. header.php - This is the header file for all pages
  4. index.php - Main leads page giving you the overview of all leads in your database.
  5. install.php - Installation script that creates the tables and variables.php file for you on first run (you can remove this after install).
  6. lead.php - Individual lead view showing all lead fields and notes pertaining to that lead.
  7. login.php - The login page
  8. logout.php - The logout page that clears sessions and redirects to the login page
  9. phpinfo.php - PHP information about your server you can view
  10. stats.php - Statistics page giving you graphical summaries of your leads and contacts
  11. topBanner.php - Top Menu items are in this file
css folder
  1. style.css - Styling for all pages, organized by page sections
  2. menu.css - Styling for the top menu.
  3. img - More image files used
js folder
  1. ajaxfileupload.js - Used in uploading leads and contacts from a spreadsheet
  2. config.js - Javascript that partains to the config.php page
  3. FlashCanvas - Used to allow displaying graphs for ie versions before 9
  4. Flotr2-master - Used to create the graphs on the statistics page
  5. jquery-1.9.1.min.js - jQuery!
  6. jquery.simplemodal-1.4.4.js - used for creating the modals
  7. jquery.ui.touch-punch.js - Javascript supporting touch screen interactions
  8. jquery-ui-1.10.3.custom.min.js - jQuery UI
  9. lead.js - Javascript pertaining to the lead.php page
  10. leads.js - Javascript pertaining to the leads.php page
  11. sha1.js - Used to hash passwords in javascript before being sent to webserver ( a little extra security)
  12. stats.js - Javascript pertaining to the stats.php page
  13. tinymce - TinyMCE used for formatting notes created
  14. utf8.js - Used in conjunction with sha1.js for client side password hashing
ajax folder
  1. ajaxFunctions.php - Most of the ajax requests use this file on the site.
  2. exportDirect.php - Exporting of csv and xlsx leads and contacts direct to browser instead of depending on file storage
  3. import.php - Ajax processing for the import tool.
  4. install.php - Ajax processing for install.php in root directory
classes folder
  1. dbClass.php - PDO database class, used for all database interaction.
  2. logging.php - Class for logging actions performed on site
  3. paging.php - Class used to create pagination links on site
  4. PHPExcel - PHPExcel library files
  5. variables.php - The configuration file containing database and other settings specific to your site
  6. variables.php.example - An example configuration file for reference
img folder
  1. Folder containing images used across LCM
